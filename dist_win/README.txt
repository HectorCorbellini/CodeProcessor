Code Processor for Windows
=======================

This is the Windows version of Code Processor, a tool that helps format code for sharing with AI platforms.

Installation Instructions
------------------------

Option 1: Quick Start (Requires Python)
---------------------------------------
1. Make sure you have Python 3.11 or higher installed on your system
   - Download from: https://www.python.org/downloads/
   - During installation, check "Add Python to PATH"

2. Double-click on "run_codeprocessor.bat" to start the application
   - This will automatically install required dependencies
   - The application will start automatically

Option 2: Using the Executable (Coming Soon)
-------------------------------------------
1. Double-click on "CodeProcessor.exe"
2. Follow the on-screen instructions

Troubleshooting
--------------
If you encounter any issues:

1. Make sure Python 3.11+ is installed and added to your PATH
2. Try running the application from the command prompt:
   - Open Command Prompt
   - Navigate to this directory
   - Run: python main.py

3. If you're missing dependencies, install them manually:
   - pip install customtkinter

For more information, visit: https://github.com/HectorCorbellini/CodeProcessor

Enjoy using Code Processor!
